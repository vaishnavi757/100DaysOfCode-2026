## 1. Plus One (LeetCode #66)
A problem that teaches array manipulation and carry handling by simulating addition on a large integer represented as an array.

Given an integer array digits where each element represents a digit of a number, increment the number by one and return the resulting array.

This problem is commonly asked in interviews and helps build concepts like:
- Carry propagation
- Array traversal
- Edge case handling
- Number simulation

which are important for mathematical and array-based problem solving.

**Your task:** Add one to the number represented by the array using an optimized approach.

### Examples

**Input:**
```
digits = [1, 2, 3]
```
**Output:**
```
[1, 2, 4]
```

---

**Input:**
```
digits = [4, 3, 2, 1]
```
**Output:**
```
[4, 3, 2, 2]
```

---

**Input:**
```
digits = [9]
```
**Output:**
```
[1, 0]
```

## 2. Number of Occurrences
A problem that teaches binary search and array traversal techniques by finding how many times a target element appears in a sorted array.

Given a sorted array arr[] and a target value, return the total number of occurrences of the target in the array.

This problem is commonly asked in interviews and helps build concepts like:
- Searching in sorted arrays
- Range finding
- Optimized traversal

which are important for efficient searching problems.

**Your task:** Find the number of occurrences of the target element using an optimized approach.

### Examples

**Input:**
```
arr[] = [1, 1, 2, 2, 2, 2, 3], target = 2
```
**Output:**
```
4
```

---

**Input:**
```
arr[] = [1, 1, 2, 2, 2, 2, 3], target = 4
```
**Output:**
```
0
```

---

**Input:**
```
arr[] = [8, 9, 10, 12, 12, 12], target = 12
```
**Output:**
```
3
```
